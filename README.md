## 🔍 Features

- 🔐 **Secure Launch** — Password-protected access using Tkinter
- 🌐 **Full-Page Screenshot** of any website via Selenium
- 🖥️ Modern GUI with custom styling
- 🧠 Auto-handles headless Chrome via ChromeDriver
- 💻 Easy `.exe` conversion for sharing on Windows


## ⚙️ Setup Instructions

### ✅ 1. Clone the Repo

```bash
git clone https://github.com/VirabhadraKhobare/Website-Screenshot-Taker.git
cd Website-Screenshot-Taker
```

### ✅ 2. Install Requirements

```bash
pip install -r requirements.txt
```

### ✅ 3. Run the App (Password Required)

```bash
python secure_launcher.py
```
